# vue-template-mini

Vue 3 template with minimal dependency

[open in codesandbox](https://codesandbox.io/p/github/qwasfun/vue-template-mini/main)

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Compile and Minify for Production

```sh
pnpm build
```
